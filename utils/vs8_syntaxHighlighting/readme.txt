Syntax Highlighting for Visual Studio 8 for .br files

1. Copy usertype.dat onto <Visual Studio 8>/Common7/IDE/ if not already present.
   Otherwise, append contents of usertype.dat onto <Visual Studio 8>/Common7/IDE/usertype.dat

2. Open Visual Studio 8, Select Tools->Options. Expand Text Editor, Select File Extension.
   Enter br in the Extension box, Choose your favorite editor, click Add and OK to close window.

3. Close Visual Studio 8 and Restart.
